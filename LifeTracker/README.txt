LifeTracker — локальний органайзер (Windows)
==========================================

✅ Функції:
- Трекер витрат у євро (€) з категоріями
- Календар подій (як Google Calendar) з нагадуваннями
- Завдання (to-do) з дедлайнами
- Нотатки
- Автозбереження у локальну базу SQLite (data/data.db)
- Системний трей + сповіщення (балуни)

Запуск на Windows (один клік):
-----------------------------
1) Розпакуйте ZIP у будь-яку папку.
2) Двічі клікніть **Start.bat**.
   - Перший запуск автоматично створить venv, встановить залежності і запустить додаток.
   - Далі просто **Start.bat** — і додаток відкриється.

Вимоги:
- Встановлений Python 3.10–3.12 (https://www.python.org/downloads/). На інсталяції поставте "Add Python to PATH".
- Інтернет потрібен лише один раз — щоб підтягнути PyQt5. Дані зберігаються лише локально.

Як зібрати .exe (необов’язково):
-------------------------------
1) В активованому середовищі venv встановіть: `pip install pyinstaller`
2) Зберіть: `pyinstaller --noconsole --name LifeTracker --onefile main.py`
3) Готовий exe буде в dist/LifeTracker.exe
